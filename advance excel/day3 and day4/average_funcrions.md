**sumproduct(): The SUMPRODUCT function multiplies corresponding values in two or more arrays (ranges) and then adds up those products.**

&nbsp;		=SUMPRODUCT(array1, \[array2], \[array3], …)





**subtotal():The SUBTOTAL function returns a subtotal in a list or database.**

**It can perform different functions like SUM, AVERAGE, COUNT, MAX, MIN, etc., and it automatically ignores hidden rows (if filtering is applied).**

&nbsp;		=SUBTOTAL(function\_num, ref1, \[ref2], …)





**average():**-average function calculate only numeric value and exclude all text or character in between specified column
