

**count()**=count only numeric value not string or text value



**counta()**= counta function count numeric value as well as text value.



**countblank()**= countblank() function is used to count blank cell in columns or in data.



**counta()+count()**= to count all record in column including numeric, character(words),andblankspace etc



**countif()**= count data based on specific single  condition syntax= countif(range,conditoin)



**countifs()**= count data based on multiple condition
